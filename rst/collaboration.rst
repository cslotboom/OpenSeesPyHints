================================================
 OpenSeesPy Collaboration through `DesignSafe`_
================================================

`DesignSafe`_ provides an environment for collaboration:

.. tip::
   * Make sure the steps in :doc:`designsaferun` are completed.

.. tip::
   * Go to ``Data Depot`` in DesignSafe.

   .. image:: /_static/datadepot.png

.. tip::
   * Select folder to be shared and click ``Share`` button.
     In the popup window, search users to share.

   .. image:: /_static/datashare.png


.. tip::
   * The other user can find the folder you shared in the ``Shared with Me``.

   .. image:: /_static/sharewithme.png
