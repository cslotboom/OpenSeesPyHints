================
 PyPi (Mac)
================

Install Python
--------------

* Install Python 3.8



Install in terminal
-------------------

* To install

   ::

      python3.8 -m pip install openseespy

      python3.8 -m pip install --user openseespy

* To upgrade

   ::

      python3.8 -m pip install --upgrade openseespy

      python3.8 -m pip install --user --upgrade openseespy

* To import

   ::

      import openseespy.opensees as ops
