======================
 OpenSeesCloud Module
======================

The ``Module`` is the building block
for ``OpenSeesCloud``. All functions,
controls, and documentations are available
in ``Modules``.


Add module
----------

To add a module,

- open ``Add Module: start by adding a Module``

  .. image:: /_static/addmodule.png
	     
- Click a ``Module`` you want to add.
- A ``Module`` with its ``id`` will appear below.

Module header
-------------

Each ``Module`` has a header, which has three parts from left to right:

- ``Module`` icon
- ``Module`` name with its ``id``
- ``Module`` operations:
  
  - move module down

    .. image:: /_static/moduledown.png

    Click this button to move the ``Module`` down

  - move module up

    .. image:: /_static/moduleup.png

    Click this button to move the ``Module`` up

  - delete module

    .. image:: /_static/moduledelete.png

    Click this button to move the ``Module`` to the :doc:`TrashCan`.


  - open module

    .. image:: /_static/moduleopen.png

    Click this button to open the ``Module`` contents.


  - close module

    .. image:: /_static/moduleclose.png

    Click this button to close the ``Module`` contents.  


Mutiple Modules
---------------

- Each ``Module`` can have multiple instances with different ``ids``.
- Only for some modules, it makes sense to have multiple instances, such as :doc:`GenericModel`.
