=======================
 Viewer Setting Module
=======================

- View Setting Module is for controlling the cloud viewer displaying.

- View Setting Module itself is a :doc:`keyboard-shortcuts`.

- The default settings are shown when the module is first time added.

- You can change any settings by changing the parameter in setting commands.

- All setting commands can be found in :doc:`SearchAll`.

- To recover the default settings, delete all contents in the module.
